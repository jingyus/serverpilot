## 核心定位

### ❌ **不是**传统 SaaS（云服务器托管）

传统 SaaS 思维（Datadog、PagerDuty）：
- 托管数据库（PostgreSQL）
- 托管基础设施（K8s、负载均衡）
- 自动备份、高可用、SLA

**问题**：用户已经有自己的服务器，不需要我们托管服务器。

### ✅ **是** AI-Driven 专业运维平台

**ServerPilot Cloud 的核心价值**：
1. **官方 AI 能力**（用户无需自己配置 API Key）
2. **智能模型路由**（自动选择最优模型，降低成本）
3. **专业 Skills 库**（AI 驱动的高级运维功能）
4. **AI 成本优化**（智能路由比自己买 API 便宜）

## 智能模型路由

### 路由策略设计

```typescript
// packages/cloud/src/ai/model-router.ts

/**
 * 智能模型路由器 — 根据任务复杂度自动选择最优模型
 */
export class ModelRouter {
  /**
   * 选择最优模型
   *
   * 策略：
   * 1. 危险操作（high/critical risk）→ Opus（需要最强推理）
   * 2. 简单查询（短对话、无命令）→ Haiku（节省成本）
   * 3. 复杂任务（默认）→ Sonnet（平衡性价比）
   * 4. 用户订阅等级（Enterprise 可强制使用 Opus）
   */
  selectModel(context: RoutingContext): ModelName {
    const { command, riskLevel, conversationLength, userPlan } = context;

    // Enterprise 用户可强制使用最强模型
    if (userPlan === 'enterprise' && context.forceOpus) {
      return 'claude-opus-4-6';
    }

    // 危险操作 → Opus（最强推理，避免误操作）
    if (riskLevel === 'critical' || riskLevel === 'high') {
      return 'claude-opus-4-6';
    }

    // 简单查询 → Haiku（快速响应，节省成本）
    // 条件：对话轮次 < 3 且无命令执行
    if (conversationLength < 3 && !command) {
      return 'claude-haiku-4-5';
    }

    // 知识库检索 → Haiku（简单语义匹配）
    if (context.isKnowledgeQuery) {
      return 'claude-haiku-4-5';
    }

    // 默认 → Sonnet（平衡）
    return 'claude-sonnet-4-5';
  }

  /**
   * 记录路由决策（用于成本分析和优化）
   */
  async logRoutingDecision(
    userId: string,
    context: RoutingContext,
    selectedModel: ModelName,
    actualCost: number
  ) {
    await db.insert(ai_routing_logs).values({
      userId,
      command: context.command,
      riskLevel: context.riskLevel,
      conversationLength: context.conversationLength,
      selectedModel,
      actualCost,
      timestamp: Date.now(),
    });
  }
}

export interface RoutingContext {
  command?: string;
  riskLevel?: RiskLevel;
  conversationLength: number;
  userPlan: 'free' | 'pro' | 'team' | 'enterprise';
  forceOpus?: boolean;
  isKnowledgeQuery?: boolean;
}
```

### 成本追踪与配额管理

```typescript
// packages/cloud/src/ai/cost-tracker.ts

/**
 * AI 成本追踪器 — 实时监控用户 AI 使用量
 */
export class CostTracker {
  /**
   * 记录 AI 调用成本
   */
  async trackAICall(userId: string, call: AICallMetrics) {
    await db.insert(ai_usage).values({
      userId,
      model: call.model,
      inputTokens: call.inputTokens,
      outputTokens: call.outputTokens,
      cost: this.calculateCost(call),
      timestamp: Date.now(),
    });
  }

  /**
   * 计算调用成本
   */
  private calculateCost(call: AICallMetrics): number {
    const pricing = MODEL_PRICING[call.model];
    return (
      (call.inputTokens * pricing.input) / 1_000_000 +
      (call.outputTokens * pricing.output) / 1_000_000
    );
  }

  /**
   * 检查用户配额
   */
  async checkQuota(userId: string): Promise<QuotaStatus> {
    const user = await getUserRepository().findById(userId);
    const quota = PLAN_QUOTAS[user.plan];

    // 免费用户：次数限制
    if (user.plan === 'free') {
      const monthlyCount = await this.getMonthlyCallCount(userId);
      return {
        allowed: monthlyCount < quota.maxCalls,
        remaining: quota.maxCalls - monthlyCount,
        limit: quota.maxCalls,
        type: 'count',
      };
    }

    // 付费用户：公平使用政策（软限制）
    const monthlyCost = await this.getMonthlyCost(userId);
    if (monthlyCost > quota.softLimit) {
      // 不拒绝请求，但发送警告
      await this.sendUsageWarning(userId, monthlyCost);
    }

    return {
      allowed: true,
      used: monthlyCost,
      softLimit: quota.softLimit,
      type: 'cost',
    };
  }
}

const PLAN_QUOTAS = {
  free: {
    maxCalls: 100,
  },
  pro: {
    softLimit: 50, // $50/月 软限制（超过发警告）
  },
  team: {
    softLimit: 200,
  },
  enterprise: {
    softLimit: 1000,
  },
};

const MODEL_PRICING = {
  'claude-haiku-4-5': {
    input: 0.25,
    output: 1.25,
  },
  'claude-sonnet-4-5': {
    input: 3.00,
    output: 15.00,
  },
  'claude-opus-4-6': {
    input: 15.00,
    output: 75.00,
  },
};
```

/**
 * AI-driven 日志巡检 Skill
 *
 * 与传统日志分析的区别：

export async function scanLogs(serverId: string): Promise<ScanReport> {
  // 1. 收集最近 1 小时的关键日志

    }
  ],

## AI Agent vs 传统 SaaS（核心理念）

### 传统 SaaS 思维（错误）

**典型例子**：AWS RDS、Heroku、Vercel

| 传统 SaaS | 说明 | 用户付费买什么 |
|----------|------|---------------|
| **托管基础设施** | 我们提供服务器、数据库、存储 | 基础设施租赁 |
| **数据在我们这里** | 用户数据存储在 SaaS 提供商 | 便利性 + 锁定 |
| **功能分级** | 免费版功能受限，付费解锁 | 功能解锁 |
| **运维是我们的事** | SaaS 提供商负责维护 | 省心省力 |

**问题**：
- ❌ 用户已经有自己的服务器，不需要我们托管
- ❌ 数据锁定（vendor lock-in）
- ❌ 功能分级导致开源版"不完整"

### AI Agent 思维（正确）— ServerPilot Cloud

**核心理念**：我们是 AI 运维专家，不是服务器托管商

| AI Agent | 说明 | 用户付费买什么 |
|----------|------|---------------|
| **AI 连接用户服务器** | 用户服务器在用户那里 | AI 智能能力 |
| **数据在用户那里** | 用户完全控制数据，无锁定 | 隐私 + 控制权 |
| **功能完全一致** | Self-Hosted 和 Cloud 核心功能 100% 相同 | AI 自动化 + 专业 Skills |
| **AI 代替人工** | AI Agent 自动巡检、分析、告警、修复 | AI 运维专家能力 |

**优势**：
- ✅ 用户数据始终在用户自己的服务器上
- ✅ Self-Hosted 和 Cloud 功能完全一致（无锁定）
- ✅ Cloud 用户付费购买的是 AI 智能，不是服务器托管
- ✅ 可随时降级到 Self-Hosted（无数据迁移）

### 具体例子对比

#### 1. 备份功能

**传统 SaaS**（Heroku、AWS RDS）：
- 我们提供备份存储空间
- 用户数据备份到我们的 S3
- 用户付费购买备份存储
- **锁定**：数据在我们这里，迁移困难

- AI 分析并生成备份策略
- AI Agent 自动执行备份到**用户自己的 S3**
- 用户付费购买 AI 智能（策略生成 + 自动执行 + 验证测试）
- **无锁定**：数据在用户自己的存储，随时可停用 Cloud

#### 2. 安全扫描功能

**传统 SaaS**（Datadog、Sentry）：
- 用户需要安装我们的 Agent
- Agent 收集数据上传到我们的平台
- 我们提供仪表盘展示
- **锁定**：数据在我们这里，依赖我们的平台

**AI Agent**（ServerPilot Cloud）：
- AI Agent 连接到用户服务器（通过 ServerPilot Agent）
- AI 分析用户服务器的安全配置（数据不离开用户服务器）
- AI 生成加固脚本，用户直接在自己服务器上执行
- **无锁定**：用户随时可以下载 AI 生成的脚本，在 Self-Hosted 版本执行

#### 3. 成本优化功能

**传统 SaaS**（CloudHealth、Spot.io）：
- 用户授权我们访问云账单

| 维度 | 传统 SaaS | ServerPilot Cloud (AI Agent) |

| **适用场景** | 无技术团队的公司 | 有服务器的技术团队 |

**ServerPilot Cloud 的定位**：
- 不是传统 SaaS（托管服务）

```typescript
// packages/cloud/src/ai/quota-manager.ts

/**
 * AI 配额管理器
 */
export class AIQuotaManager {
  /**
   * 检查用户配额
   */
  async checkQuota(userId: string, tenantId: string): Promise<QuotaCheckResult> {
    const tenant = await getTenant(tenantId);
    const quota = PLAN_QUOTAS[tenant.plan];

    // Free 用户：次数限制
    if (tenant.plan === 'free') {
      const monthlyCount = await this.getMonthlyCallCount(userId);

      if (monthlyCount >= quota.maxCalls) {
        return {
          allowed: false,
          reason: `已达到 Free 计划限制（${quota.maxCalls} 次/月）`,
          remaining: 0,
          upgradeUrl: '/billing?upgrade=pro',
        };
      }

      return {
        allowed: true,
        remaining: quota.maxCalls - monthlyCount,
      };
    }

    // 付费用户：公平使用政策（软限制）
    const monthlyCost = await this.getMonthlyCost(userId);

    if (monthlyCost > quota.softLimit) {
      // 不拒绝请求，但发送警告
      await this.sendUsageWarning(userId, {
        used: monthlyCost,
        limit: quota.softLimit,
      });
    }

    return {
      allowed: true,
      remaining: quota.softLimit - monthlyCost,
    };
  }

  /**
   * 记录 AI 调用
   */
  async trackAICall(userId: string, call: AICallMetrics) {
    await db.insert(ai_usage).values({
      userId,
      tenantId: call.tenantId,
      model: call.model,
      inputTokens: call.inputTokens,
      outputTokens: call.outputTokens,
      cost: this.calculateCost(call),
      timestamp: Date.now(),
    });

    // 实时检查配额（防止滥用）
    await this.checkQuota(userId, call.tenantId);
  }

  /**
   * 获取本月调用次数
   */
  private async getMonthlyCallCount(userId: string): Promise<number> {
    const startOfMonth = new Date();
    startOfMonth.setDate(1);
    startOfMonth.setHours(0, 0, 0, 0);

    const count = await db
      .select({ count: sql<number>`count(*)` })
      .from(ai_usage)
      .where(
        and(
          eq(ai_usage.userId, userId),
          gte(ai_usage.timestamp, startOfMonth.getTime())
        )
      );

    return count[0]?.count || 0;
  }

  private calculateCost(call: AICallMetrics): number {
    const pricing = MODEL_PRICING[call.model];
    return (
      (call.inputTokens * pricing.input) / 1_000_000 +
      (call.outputTokens * pricing.output) / 1_000_000
    );
  }
}

const PLAN_QUOTAS = {
  free: {
    maxCalls: 100, // 硬限制
  },
  pro: {
    maxCalls: 2000, // 软限制（超过发警告，不拒绝）
    softLimit: 50, // $50/月
  },
  team: {
    softLimit: 200, // $200/月
  },
  enterprise: {
    softLimit: 1000, // $1000/月
  },
};

const MODEL_PRICING = {
  'claude-haiku-4-5': { input: 0.25, output: 1.25 },
  'claude-sonnet-4-5': { input: 3.00, output: 15.00 },
  'claude-opus-4-6': { input: 15.00, output: 75.00 },
};
```

// packages/cloud/src/api/middleware/check-ai-quota.ts

export function checkAIQuota() {
  return async (c: Context, next: Next) => {
    const { userId, tenantId } = c.get('auth');

    const quotaManager = getAIQuotaManager();
    const result = await quotaManager.checkQuota(userId, tenantId);

    if (!result.allowed) {
      return c.json({
        error: {
          code: 'QUOTA_EXCEEDED',
          message: result.reason,
          upgradeUrl: result.upgradeUrl,
        },
      }, 429);
    }

    // 将剩余配额信息添加到 response header
    c.header('X-Quota-Remaining', result.remaining.toString());

    await next();
  };
}

// packages/cloud/src/billing/stripe-integration.ts

import Stripe from 'stripe';

const stripe = new Stripe(process.env.STRIPE_SECRET_KEY!);

/**
 * 创建订阅（用户升级到 Pro/Team/Enterprise）
 */
export async function createSubscription(
  userId: string,
  tenantId: string,
  plan: 'pro' | 'team' | 'enterprise'
) {
  // 1. 创建或获取 Stripe Customer
  const user = await getUser(userId);
  let customerId = user.stripeCustomerId;

  if (!customerId) {
    const customer = await stripe.customers.create({
      email: user.email,
      metadata: { userId, tenantId },
    });
    customerId = customer.id;
    await updateUser(userId, { stripeCustomerId: customerId });
  }

  // 2. 创建订阅
  const priceId = PLAN_PRICE_IDS[plan]; // Stripe Price ID

  const subscription = await stripe.subscriptions.create({
    customer: customerId,
    items: [{ price: priceId }],
    payment_behavior: 'default_incomplete',
    payment_settings: { save_default_payment_method: 'on_subscription' },
    expand: ['latest_invoice.payment_intent'],
    metadata: { tenantId, plan },
  });

  // 3. 存储订阅信息到数据库
  await createSubscriptionRecord({
    tenantId,
    userId,
    plan,
    status: 'incomplete',
    stripeSubscriptionId: subscription.id,
    currentPeriodStart: subscription.current_period_start * 1000,
    currentPeriodEnd: subscription.current_period_end * 1000,
  });

  // 4. 返回 clientSecret（用于前端确认支付）
  const invoice = subscription.latest_invoice as Stripe.Invoice;
  const paymentIntent = invoice.payment_intent as Stripe.PaymentIntent;

  return {
    subscriptionId: subscription.id,
    clientSecret: paymentIntent.client_secret,
  };
}

 * Stripe Webhook 处理

export async function handleStripeWebhook(
  payload: Buffer,

  const event = stripe.webhooks.constructEvent(
    payload,
    signature,
    process.env.STRIPE_WEBHOOK_SECRET!
  );

  switch (event.type) {
    // 订阅激活
    case 'customer.subscription.created':
    case 'customer.subscription.updated':
      await handleSubscriptionUpdated(event.data.object as Stripe.Subscription);
      break;

    // 订阅取消
    case 'customer.subscription.deleted':
      await handleSubscriptionCanceled(event.data.object as Stripe.Subscription);
      break;

    // 支付失败
    case 'invoice.payment_failed':
      await handlePaymentFailed(event.data.object as Stripe.Invoice);
      break;

    // 支付成功
    case 'invoice.payment_succeeded':
      await handlePaymentSucceeded(event.data.object as Stripe.Invoice);
      break;
  }
}

async function handleSubscriptionUpdated(subscription: Stripe.Subscription) {
  const tenantId = subscription.metadata.tenantId;
  const plan = subscription.metadata.plan as 'pro' | 'team' | 'enterprise';

  // 更新 Tenant 计划
  await updateTenant(tenantId, {
    plan,
    maxServers: PLAN_LIMITS[plan].maxServers,
    maxUsers: PLAN_LIMITS[plan].maxUsers,
  });

  // 更新订阅状态
  await updateSubscription(subscription.id, {
    status: subscription.status,
    currentPeriodStart: subscription.current_period_start * 1000,
    currentPeriodEnd: subscription.current_period_end * 1000,
  });
}

async function handleSubscriptionCanceled(subscription: Stripe.Subscription) {
  const tenantId = subscription.metadata.tenantId;

  // 降级到 Free 计划
  await updateTenant(tenantId, {
    plan: 'free',
    maxServers: 1,
    maxUsers: 1,
  });

  // 发送通知邮件
  await sendSubscriptionCanceledEmail(tenantId);
}

/**
 * Cloud Agent 认证（增强版）
 *
 * 增加 Tenant 隔离检查
 */
export async function authenticateCloudAgent(
  serverId: string,
  agentToken: string
): Promise<CloudAuthResult> {
  // 1. 验证 agentToken
  const server = await getServerByToken(agentToken);
  if (!server) {
    throw new Error('Invalid agent token');
  }

  // 2. 检查服务器是否属于有效的 Tenant
  if (!server.tenantId) {
    throw new Error('Server not associated with any tenant');
  }

  const tenant = await getTenant(server.tenantId);
  if (!tenant) {
    throw new Error('Tenant not found');
  }

  // 3. 检查 Tenant 订阅状态
  if (tenant.subscriptionStatus === 'past_due') {
    throw new Error('Subscription expired. Please update billing.');
  }

  // 4. 检查服务器数量限制
  const serverCount = await getServerCountByTenant(tenant.id);
  if (serverCount > tenant.maxServers) {
    throw new Error(`Server limit exceeded (${tenant.maxServers} max)`);
  }

  return {
    server,
    tenant,
    userId: server.userId,
    permissions: await getUserPermissions(server.userId, tenant.id),
  };
}

/**
 * Cloud 用户注册流程
 *
 * 与 Self-Hosted 的区别：
 * - Self-Hosted: 注册 → 创建用户 → 成为 owner
 * - Cloud: 注册 → 创建 Tenant → 创建用户 → 成为 Tenant owner
 */
export async function cloudRegister(data: {
  email: string;
  password: string;
  name?: string;
  companyName?: string;
}) {
  // 1. 验证邮箱唯一性
  const existing = await getUserByEmail(data.email);
  if (existing) {
    throw new Error('Email already registered');
  }

  // 2. 创建 Tenant（以公司名或用户名命名）
  const tenantName = data.companyName || data.name || data.email.split('@')[0];
  const tenantSlug = generateSlug(tenantName); // 唯一 slug（用于子域名）

  const tenant = await createTenant({
    name: tenantName,
    slug: tenantSlug,
    plan: 'free', // 默认 Free 计划
    maxServers: 1,
    maxUsers: 1,
  });

  // 3. 创建用户（关联到 Tenant）
  const user = await createUser({
    email: data.email,
    password: hashPassword(data.password),
    name: data.name,
    tenantId: tenant.id,
    role: 'owner', // Tenant 的 owner
  });

  // 4. 更新 Tenant 的 ownerId
  await updateTenant(tenant.id, { ownerId: user.id });

  // 5. 创建默认团队设置
  await createDefaultTeamSettings(tenant.id);

  // 6. 发送欢迎邮件
  await sendWelcomeEmail(user.email, {
    tenantSlug,
    dashboardUrl: `https://${tenantSlug}.serverpilot.io`, // 子域名
  });

  // 7. 返回 JWT token
  return {
    user,
    tenant,
    tokens: generateTokens(user.id, tenant.id),
  };
}

// packages/cloud/src/api/middleware/verify-tenant.ts

export function verifyTenant() {
  return async (c: Context, next: Next) => {
    const { tenantId: authTenantId } = c.get('auth'); // JWT 中的 tenantId
    const headerTenantId = c.req.header('X-Tenant-ID'); // Header 中的 tenantId

    // 防止用户伪造 tenantId
    if (headerTenantId && headerTenantId !== authTenantId) {
      return c.json({
        error: {
          code: 'TENANT_MISMATCH',
          message: 'Tenant ID mismatch',
        },
      }, 403);
    }

    await next();
  };
}
