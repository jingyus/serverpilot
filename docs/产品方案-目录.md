## 核心定位

**ServerPilot** 是一款**开源的** AI 驱动服务器运维平台。

- **一句话**: "跟 AI 聊天就能管服务器 — 开源、安全、自主可控"
- **市场定位**: 宝塔面板的 AI 时代替代品
- **核心理念**: `用户 → 对话 AI → AI 生成计划 → Agent 执行 → 结果反馈`

### 三大组件

- **Server** (大脑) = 理解意图、制定计划、调度执行
- **Agent** (手脚) = 探测环境、执行命令、汇报状态
- **AI Provider** (智能引擎) = 可插拔模型 (Claude/GPT/DeepSeek/Ollama)

---

## MVP 范围 (v0.1 - 6周)

**目标**: 完成"自部署 → 安装 Agent → 连接 → 对话运维"的基本闭环

### ✅ 核心功能

| 模块 | MVP 范围 |
|------|---------|
| **部署方式** | `docker compose up` 一键自部署 |
| **AI Provider** | Claude + OpenAI + Ollama (三选一) |
| **Agent** | 安装、密钥生成、WSS连接、系统探测、命令执行 |
| **Dashboard** | 服务器列表、密钥连接、基本监控、对话界面 |
| **AI 对话** | 单服务器对话、命令生成、实时执行反馈 |
| **服务器档案** | 基础档案 (系统信息 + 软件清单 + 服务状态) |
| **安全** | 命令分级白名单、参数审计、用户确认 |
| **知识库** | 内置知识库 (10+ 常见软件) + RAG 检索注入 |

### ❌ MVP 不做

- 定时任务、团队协作、告警通知、批量操作
- 快照回滚 (v0.2)
- 文档自动抓取 (v0.2)
- 云版 (v1.0)

## 技术栈概要

### 后端
- **运行时**: Node.js 22+
- **语言**: TypeScript 5.7+
- **框架**: Hono 4.7+ (Web) + ws 8.18+ (WebSocket)
- **数据库**: SQLite (社区版) + Drizzle ORM
- **AI SDK**: @anthropic-ai/sdk, openai
- **验证**: Zod 3.25+

### 前端
- **框架**: React 18 + Vite 5
- **路由**: React Router 6
- **状态**: Zustand 5
- **UI**: Shadcn/ui + Tailwind CSS 3
- **图表**: Recharts 2

### Agent
- **运行时**: Bun (编译为单一二进制)
- **CLI UI**: @clack/prompts
- **通信**: ws (WebSocket)

### 基础设施
- **容器化**: Docker + Docker Compose
- **CI/CD**: GitHub Actions
- **测试**: Vitest + Playwright
- **部署**: Fly.io (云版)

## 开发优先级

### 第一优先级 (MVP v0.1 核心功能)

1. **基础设施**
   - Docker Compose 部署配置
   - SQLite 数据库 Schema + Drizzle ORM
   - 配置管理、日志系统

2. **Agent 核心**
   - 安装脚本 + 二进制编译
   - WSS 连接 + 认证
   - 系统探测 + 命令执行

3. **Server 核心**
   - WebSocket 服务器
   - AI Provider 适配器 (Claude/OpenAI/Ollama)
   - 命令分级 + 参数审计
   - 基础服务器档案

4. **Dashboard 基础**
   - 服务器列表页
   - 对话界面 (Chat)
   - Agent 连接管理

5. **知识库**
   - 内置知识库加载
   - RAG 检索 + Prompt 注入

### 第二优先级 (v0.2 增强功能)

- 快照回滚
- 完整服务器档案
- 定时任务
- 基本监控图表
- 邮件告警
- 知识库自动学习

### 第三优先级 (v0.3 开源准备)

- 文档完善
- CI/CD 自动化
- 一键安装脚本
- 更多 AI Provider
